# Fintech
![Badge em Desenvolvimento](http://img.shields.io/static/v1?label=STATUS&message=EM%20DESENVOLVIMENTO&color=GREEN&style=for-the-badge)

Este projeto é uma Fintech em desenvolvimento, inspirado pelos ensinamentos da FIAP. A aplicação visa facilitar a organização financeira pessoal, oferecendo ferramentas intuitivas e eficientes.


## Tecnologias Utilizadas
Até o momento, as seguintes tecnologias foram empregadas na criação das páginas:
- HTML
- CSS
- JavaScript
- Tailwind CSS


## Etapas do Desenvolvimento
O desenvolvimento do projeto passou pelas seguintes etapas:

- Story Mapping
  > Mapeamento das histórias dos usuários para definir as funcionalidades principais.
- Documento de Visão do Sistema
  > Elaboração de um documento que detalha a visão geral do sistema.
- Criação do Layout e Prototipagem
  > Desenvolvimento do layout e protótipos das interfaces.
- Modelo Lógico e Físico
  > Construção do modelo lógico e físico do banco de dados.
- Implementação das Telas
  > Codificação das telas utilizando as tecnologias mencionadas.


## Status Atual
O projeto está em fase de desenvolvimento ativo, e muitas atualizações e melhorias ainda serão implementadas.
